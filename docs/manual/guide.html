User Guide

AI Lawsuit Radar의 설정 및 사용 방법에 대한 상세 가이드입니다.

1. 프로젝트 개요

AI Lawsuit Radar는 인공지능 학습 데이터와 관련된 저작권 소송을 자동으로 탐색하고 분석하는 도구입니다. 전 세계 법원 데이터와 뉴스 피드를 실시간으로 모니터링하여 핵심적인 법적 리스크를 리포트합니다.

2. 시작하기 (Quick Start)

GitHub Actions를 통해 서버 없이 무료로 운영할 수 있습니다.

  • Step 1: 본 저장소를 Fork 합니다.
  • Step 2: Settings > Secrets and variables > Actions 메뉴로 이동합니다.
  • Step 3: 아래의 필수 환경 변수를 등록합니다.
GEMINI_API_KEY: Google AI Studio에서 발급받은 API 키
SLACK_WEBHOOK_URL: 알림을 받을 Slack Webhook URL
GITHUB_TOKEN: Issue 발행 권한이 있는 Personal Access Token

3. 주요 기능 상세

의미론적 중복 제거 (Semantic Deduplication)

단순히 제목이 같은 뉴스를 제외하는 것이 아니라, Gemini Embedding 모델을 사용하여 내용이 유사한 정보를 자동으로 걸러냅니다. 이 기능은 GEMINI_SEMANTIC_DEDUP = 1 설정을 통해 활성화할 수 있습니다.
