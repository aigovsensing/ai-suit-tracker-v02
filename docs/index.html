Capabilities

Real-time Monitoring & Analysis

단순한 데이터 수집을 넘어 법률적 맥락을 파악하고 정밀한 리스크 점수를 산출합니다.

Legal Intelligence

미국 연방법원(PACER) 데이터를 실시간 조회하여 AI 학습 관련 최신 고소장을 즉각 수집하고 분석합니다.

Gemini AI Powered

Google Gemini 1.5 Pro를 통해 복잡한 법률 본문을 핵심 요약하고 의미론적 중복을 완벽하게 필터링합니다.

Risk Scoring

저작권(820) 코드와 'training' 키워드 가중치를 조합하여 0-100점 사이의 정밀한 리스크 등급을 제공합니다.

Visual Reporting

Imagen 3 기술을 활용하여 일일 리포트의 핵심 내용을 지브리 풍 일러스트로 시각화하여 제공합니다.

Architecture

How it Works

데이터 수집부터 지능형 분석, 리포팅까지의 기술적 아키텍처입니다.

01

Ingestion & Discovery

CourtListener API와 RSS Feed를 통해 최신 소송 및 뉴스 정보를 1시간 주기로 탐색합니다.

02

Semantic Processing

Gemini Embedding 모델을 활용하여 기존 리포트와의 유사도를 분석, 중복 정보를 제거합니다.

03

Multi-Channel Dispatch

정제된 정보는 GitHub Issues와 Slack을 통해 즉각적으로 배포됩니다.

Setup

Enterprise Ready Setup

# AI 지능형 기능 설정 (Secrets/Variables)
GEMINI_API_KEY = "your_key"
GEMINI_SEMANTIC_DEDUP = 1
GEMINI_DAILY_REPORT_IMAGEGEN = 1
GITHUB_TOKEN = "ghp_your_token"
SLACK_WEBHOOK = "https://hooks.slack..."
